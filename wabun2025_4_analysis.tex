\section{考察}

\subsection{シミュレータ回路の構成}
\label{sec:circuit-structure-analysis}
\subsubsection{2つの回路方式の資源消費の比較}
図 \ref{fig:hamiltonian-gatecount} に示したように、必要とするqubit数は算術演算回
路とUCRZのいずれも、座標のqubit数に対して線形に増加するが、算術演算の方が定数
項・傾き共に大きい。一方、回路の深さの尺度として2qubitゲートの数を見ると、
UCRZゲートは入力（次元数×座標qubit数）に対して指数的にゲート数が増大するのに対し、
算術演算回路は緩やかな増加にとどまる。しかし、小規模なモデルにおいてはUCRZゲートの
定数項の小ささが支配的となる。したがって、小規模・低次元の系ではUCRZゲートによる
直接的な状態生成が有利であり、将来的に大規模・高解像度のシミュレーションを行う
際には算術演算が深さの面で優位になるトレードオフが確認された。

\subsubsection{量子エミュレータにおけるUCRZゲートの有用性}
古典コンピュータ上の量子エミュレータにおいては、シミュレート可能な状態ベクトル空
間がハードウェアのメモリ（RAM）上限によって指数関数的に制限される（通常
20〜30qubit程度）。今回検討した2つの回路方式のうち、UCRZゲートを用いてアンシラ
qubitの消費を抑える手法は、この厳しいqubit数の制約を回避し、エミュレータ上でより
高い次元（本研究における2次元水素原子モデルなど）のハミルトニアンシミュレーショ
ンを実行可能にするという点で、アルゴリズムの検証に極めて有用である。

\subsubsection{FTQCデバイスへの適用を見据えた実装コストの展望}
エミュレータ上ではどの演算ゲートも一律のコストで実行可能であるが、実際の誤り耐性
量子コンピュータ（FTQC）においては事情が異なる。特に有力視されているSurface code
等の誤り訂正方式では、Cliffordゲートに比べて非Cliffordゲートの実行にマジック状態
蒸留等の莫大な物理リソースを要する。算術演算回路におけるCC-NOTゲートや、UCRZゲー
トにおける任意のRz回転ゲートはまさにこれに該当し、最終的にはこれらをいくつのT
ゲート（T-count / T-depth）に近似・分解できるかがボトルネックとなる。特にUCRZ
ゲートのRz回転は、要求する精度に応じてTゲートの消費量が大きく変動するため、実機
への適用にあたっては、誤り訂正のオーバーヘッドを踏まえた上で両方式のゲート実装コ
ストを定量的に比較検討していくことが今後の課題となる。

\subsection{クーロンポテンシャルの極の扱い}
\label{sec:coulomb-potential-handling-analysis}
\subsubsection{実効半径が$\deltar$/4である理由}
図 \ref{fig:Ha1-vs-delta1} に示すように、局所平均ポテンシャルで用いる実効半径
$\reff=\deltar/a$ の最適値は2次元モデルにおいては、$a=4$の時に得られた。すなわ
ち、$\reff=\deltar/4$と設定した際に、クーロンポテンシャルの極（特異点）による発
散を最も効果的に正則化できた。この結果は、当初の直観とは異なる興味深いものであっ
た。

当初、我々は特異点を中心とする半径$\deltar/2$の円領域内におけるエネルギー期待値
$\braket{\psi|H|\psi}$に着目し、解析解を用いた積分値と代表値を用いた近似式が一致
するように$\reff$を定めた。この方法で導出される$\reff$は格子間隔$\deltar$の高次
項を含んでおり、極限$\deltar\rightarrow0$において$\reff=\deltar/4$に漸近する。
座標分解能が粗い条件（例えば$\nb=6$）ではこの高次項が無視できないと想定し、高次項
を含んだ$\reff$を用いて時間発展シミュレーションを行った。しかし比較検証の結果、
高次項を切り捨てた$\deltar/4$を用いた場合の方が、系のエネルギー値が解析解に近づ
くことが判明した。

この要因を探るべく、波動関数による重みづけを行わず、単なるポテンシャル関数
$V(r)=-1/r$の幾何学的な領域内平均が代表値と一致するように$\reff$を再評価したとこ
ろ、高次の項を含まない$\reff=\deltar/4$が直接的に導出された。この事実は、有限に
離散化されたグリッド空間においては、波動関数のグリッド内での高次の変化を仮定する
よりも、特異点となる格子点が代表する微小領域の単純な平均場をポテンシャルとして採
用する方が物理的に妥当であることを示唆している。

\subsubsection{円領域内近似の妥当性}

本来、格子点を中心とする局所平均は、その格子点が代表する一辺$\deltar$の正方形領
域で積分されるべきであるが、本研究では、解析的に閉じた式(Closed-form expression)
を得るために、特異点（格子点）を中心とする半径$\deltar/2$の円形領域で計算を行っ
た。クーロンポテンシャルの積分値は特異点である原点近傍での寄与が支配的であり、か
つ等方性を持つ。そのため、相対的にポテンシャルが小さくなる正方形の四隅を切り捨て
た円形近似であっても本質的な空間スケーリングを十分に捉えられていると考えられる。
実際に数値実験におけるエネルギー値がこの近似から得られた値($a=4$)で最適となった
ことが、本近似の妥当性を裏付けている。

\subsubsection{高次項の切り捨てが精度を向上させる理由}
前述の通り、$\braket{\psi|H|\psi}$を用いた厳密な評価では$\deltar$の高次項が現れる
が、これをシミュレーションに適用するとかえって精度が低下した。これは、有限の格子
間隔$\deltar$を持つ粗いグリッド上では、格子点周辺における波動関数の高次な空間変
化（曲率）を表現しきれないためだと推測される。結果として、グリッドの解像度を超え
た波動関数の微細な形状変化に依存しない、単純な幾何学的平均（高次項の切り捨て）を
採用することが、最も安定した正則化パラメータを与えることが明らかになった。

\subsection{誤差を最小にするシミュレーションパラメータ}
\label{sec:parameter-selection-for-minimum-error-analysis}
\subsubsection{所与の$\nb$におけるシミュレーションのダイナミックレンジ}
一般に、シミュレーションにおいて扱う対象の系は異なる量子数を持つ複数の固有状態の
線形結合で表され、これらは単一の空間グリッド間隔$\deltar$のもとで同時に表現され
る必要がある。

図\ref{fig:dq-tradeoff} に示した $\deltar$ に対する空間離散化誤差と領域打切り誤
差（カットオフ誤差）のトレードオフ分析を踏まえると、最も小さな$\deltar$を必要と
するのは、波動関数の裾野が最も狭い基底状態である。基底状態と同時になるべく高い励
起状態まで表現しようとする場合、$\deltar$の値は、基底状態の表現に支障がない範囲
でできるだけ大きく設定することが望ましい。その際、カットオフ誤差の影響を受けずに
表現可能な最大の量子数が決まり、基底状態からその最大の量子数までの範囲が、本シ
ミュレーションの表現可能な範囲、すなわち「ダイナミックレンジ」となる。これは、限
られた量子リソースでシミュレーションを実行する際の、表現可能な状態空間を事前に見
積もるための強力な指針となる。

\subsubsection{$\nb$の増加と最適パラメータのプラトー}
図\ref{fig:dq-tradeoff}を見ると、$\nb$が増加するにつれて、離散化誤差とカットオフ
誤差の双方が無視できる$\deltar$のプラトー（平坦な領域）が拡大することが確認され
た。これは、利用可能なqubit数が増加することで、系がパラメータ設定誤差に対してロ
バストになることを意味する。ハードウェアが未成熟で$\nb$が少ない環境では厳密な
$\deltar$のチューニングが必要であるが、将来的に$\nb$が増えれば、シミュレーション
の精度がパラメータ選択に対してより寛容になることが期待される。

\subsubsection{Trotter誤差の最悪ケースの誤差予想と計算上の誤差の比較}
従来のTrotter誤差評価は演算子ノルムに基づく最悪ケースを想定するため、極端に小さ
な$\deltat$を要求する傾向がある\cite{Burgarth2024.PhysRevResearch.6.043155}。し
かし、基底状態付近の固有状態にエネルギーが集中しているような現実的な系において
は、状態に依存した誤差評価がより適切である。本実験の結果は、形式的なTrotter誤差
の要請よりも粗い、サンプリング定理に基づく位相変化の限界（半周期条件）を満たす
$\deltat$を用いることで、所与の$\nb$における精度の限界に近い結果が得られることを
示している。これにより、無駄に時間ステップを細かくすることによる量子回路の深さの
増大を回避できる。

\subsubsection{空間誤差と時間誤差のトレードオフ}
図 \ref{fig:trotter-error-t1-part1} に示したように、 $\deltat$をある閾値以下に細
かくしても、系のエネルギー誤差の減少が飽和する現象が観測された。これは、シミュ
レーション全体の支配的な誤差要因が、時間誤差（Trotter誤差やサンプリング誤差）か
ら、空間グリッド$\deltar$の粗さに起因する空間誤差（主として空間離散化誤差）へと
支配要因がシフトしたためであると解釈できる。すなわち、ある$\nb$と$\deltar$の組み
合わせで到達可能な空間精度には限界があり、それと不釣り合いに$\deltat$を細かく
（すなわち量子回路を深く）することは量子リソースの浪費に繋がる。本分析は、所与の
qubit数$\nb$に対して空間分解能$\deltar$と時間ステップ$\deltat$をバランス良く決定
するための統合的な手順を提供するものである。
